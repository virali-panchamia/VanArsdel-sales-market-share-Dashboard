# VanArsdel-sales-market-share-Dashboard
Developing dashboard end to end from data import, transformation, modeling, and visualization using VanArsdel's sales and market share dataset.
Used Power BI

# Key skills demonstrated
- Power Query data transformation (splitting columns, conditional columns, fill down)
- Data modeling (relationships, cardinality, DAX calculated columns/measures)
- DAX (SAMEPERIODLASTYEAR, CALCULATE, DIVIDE for % growth)
- Report design (drill-down hierarchies, cross-filtering, slicers, matrix visuals)
